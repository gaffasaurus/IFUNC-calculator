IFUNC Calculator

Separate pitches by spaces (ex. A B C# D etc...). Click the "+" button to add more entries.

Note: The interval is measured from Collection 1 to Collection 2. For example, if you input "C" in the first box and "E" in the second box, it will return "4".

Collection 1's input will populate Column 1 top to bottom, and Collection 2's input will populate Row 1 left to right.
